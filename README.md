# УПК РФ — 100 задач (Flutter Android App)

Учебное приложение для подготовки к экзамену по алгоритмам назначения судебных заседаний (УПК РФ).

## Что внутри

- **100 задач** по 8 категориям:
  - Подсудность (10 задач)
  - Основания для предварительного слушания (15 задач)
  - Сроки — не под стражей (15 задач)
  - Сроки — под стражей (15 задач)
  - Мировой судья (10 задач)
  - Поиск ошибок в постановлениях (17 задач)
  - Составление постановления (10 задач)
  - Комплексные задачи (8 задач)

- Три уровня сложности: лёгкая / средняя / сложная
- Кнопка «Показать ответ» с подробным разбором
- Отметка выполненных задач (сохраняется на устройстве)
- Поиск и фильтрация по категориям
- Прогресс-бар на главном экране

## Установка Flutter (один раз)

### Windows
1. Скачайте Flutter SDK: https://docs.flutter.dev/get-started/install/windows
2. Распакуйте в `C:\flutter`
3. Добавьте `C:\flutter\bin` в PATH (Система → Переменные среды)
4. Установите Android Studio: https://developer.android.com/studio
5. В Android Studio: SDK Manager → установите Android SDK (API 34)
6. Запустите: `flutter doctor` — все ✓ зелёные

### macOS
```bash
brew install --cask flutter
flutter doctor
```

## Сборка APK

```bash
# 1. Перейдите в папку проекта
cd upk_app

# 2. Загрузите зависимости
flutter pub get

# 3. Соберите APK
flutter build apk --release

# APK будет здесь:
# build/app/outputs/flutter-apk/app-release.apk
```

## Установка на телефон

### Способ 1: USB
```bash
# Включите режим разработчика на Android:
# Настройки → Об устройстве → нажмите 7 раз "Номер сборки"
# Включите "Отладка по USB"

flutter install
```

### Способ 2: Вручную
1. Скопируйте `app-release.apk` на телефон
2. Откройте файл на телефоне
3. Разрешите установку из неизвестных источников

## Структура проекта

```
lib/
  main.dart                    — точка входа
  models/task.dart             — модель задачи
  data/tasks_data.dart         — все 100 задач
  screens/
    home_screen.dart           — главный экран
    task_list_screen.dart      — список задач с фильтрами
    task_detail_screen.dart    — экран задачи с ответом
  services/progress_service.dart — сохранение прогресса
```

## Правовые нормы в задачах

| Норма | Содержание |
|-------|------------|
| ст. 227 УПК РФ | Сроки вынесения постановления (30/14 суток) |
| ст. 229 УПК РФ | Основания для предварительного слушания |
| ст. 231 УПК РФ | Назначение судебного заседания (5 суток уведомление) |
| ст. 233 УПК РФ | Начало разбирательства (7 суток от вручения; мировой судья 3-14 суток) |
| ст. 234 УПК РФ | Предварительное слушание (единолично, закрытое, 3 суток) |
| ст. 241 УПК РФ | Основания для закрытого заседания |
